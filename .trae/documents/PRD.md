## 1. 产品概述

面向千万级标箱集装箱港口的高智能化无人调度大屏系统，基于真实工业重型装备数据流驱动，构建自动化堆场全景三维数字孪生平台。系统通过 Modbus TCP 实时采集场桥 PLC 寄存器数据，解析吊具与轨道吊（RMG）的三维空间绝对坐标及电流负载，以 50Hz 刷新率通过 WebSocket 推送至前端，前端利用 Three.js InstancedMesh 技术对几十万规格相同的集装箱进行一次性实例化批次渲染，并根据堆位码（Bay-Row-Tier）动态调整矩阵位置，实现港口堆场的全景实时监控与智能调度。

- **目标用户**：港口调度中心操作员、堆场管理决策层、设备维护工程师
- **核心价值**：将分散的 PLC 数据融合为统一的三维可视化界面，消除人工巡检盲区，提升堆场作业效率与设备安全性

## 3. 核心流程

系统启动后，Node.js 工控网关通过 Modbus TCP 协议轮询变电站与场桥 PLC 寄存器，按位解析出吊具和 RMG 的三维空间物理绝对坐标与电流负载数据，通过 WebSocket 以 50Hz 刷新率推送到前端。前端 React 应用加载超大范围 GLTF 堆场模型，利用 InstancedMesh 批量渲染几十万集装箱，并根据实时推送的 Bay-Row-Tier 堆位码动态调整每个集装箱的矩阵位置。RMG 与吊具的位置同步更新到三维场景中对应模型实例。

```mermaid
    "寄存器按位解析" --> "RMG 坐标 + 电流负载"

    "寄存器按位解析" --> "堆位码 Bay-Row-Tier"

    "吊具坐标 + 负载数据" --> "WebSocket 50Hz 推送"

    "RMG 模型位置同步" --> "三维场景渲染"
```

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深海蓝（#0A1628）为底色，科技青（#00E5FF）为主强调色，琥珀橙（#FF9100）为告警/重点色
- **次色调**：钢铁灰（#2A3A4A）面板背景，冰蓝（#80D8FF）次要数据，暗红（#FF1744）紧急告警
- **按钮风格**：扁平化 + 微光边框，悬停时发光扩散效果
- **字体**：标题使用 Rajdhani（工业科技感），数据使用 JetBrains Mono（等宽数字），正文使用 Noto Sans SC
- **布局风格**：全屏沉浸式大屏布局，浮窗式信息面板，半透明毛玻璃效果叠加层
- **图标风格**：线性描边图标，2px 描边宽度，与科技青主题统一

### 4.3 响应式设计

- 桌面端优先，目标分辨率 3840×2160（4K 大屏）及 1920×1080
- 三维场景始终全屏，信息面板可折叠/展开
- 面板支持磁吸布局，可拖拽重排

### 4.4 三维场景设计

- **环境/HDRI**：港口夜间工业风格，低角度冷色调 HDRI，微弱月光 + 区域灯光
- **灯光设置**：1 个方向光模拟月光（强度 0.3，冷蓝色），4 个区域点光源模拟堆场作业灯（暖白色，强度 0.8），环境光强度 0.15
- **摄像机设置**：默认鸟瞰 45° 俯视，正交 + 透视模式切换，最小可视范围 500m × 500m
- **构图与焦点**：堆场中心为视觉焦点，RMG 运动路径为引导线，集装箱矩阵形成视觉纵深
- **交互与动画**：点击集装箱弹出堆位信息浮窗，双击 RMG 切换跟随视角，鼠标滚轮缩放，拖拽旋转
- **后处理效果**：泛光（Bloom）用于高亮选中和告警元素，环境遮蔽（SSAO）增加集装箱层次感，色彩校正增强科技感
- **性能预算**：目标 60fps 渲染，InstancedMesh 单次 Draw Call 渲染 ≤30 万实例，LOD 切换阈值 200m/500m
